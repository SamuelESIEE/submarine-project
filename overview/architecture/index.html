    System Architecture

    This diagram illustrates the main subsystems of the autonomous submarine.

    ```mermaid graph TD A[Main System Controller] --> B[Sensor Suite] A --> C[Motor Control System] A --> D[Communication Module] A --> E[Power Management System] A --> F[Data Storage] A --> G[Backup Controller]

    B --> B1[IMU] B --> B2[Depth Sensor] B --> B3[Temperature Sensors] B --> B4[Leak Detector] B --> B5[Battery Voltage/Current Sensor]

    C --> C1[Thrusters] C --> C2[ESC / Motor Drivers]

    D --> D1[Radio Module] D --> D2[Debug USB] D --> D3[Optional GPS]

    E --> E1[Lithium Battery Pack] E --> E2[Voltage Regulation] E --> E3[Piezo Backup Unit]

    G --> G1[Wake-up Logic] G --> G2[Pressure Monitor] G --> G3[Emergency Surface Trigger]
